# 🔥 Firebase 연동 가이드

## 1단계: Firebase 프로젝트 생성

1. Firebase Console 접속

```
https://console.firebase.google.com/
```

2. Google 계정으로 로그인
3. "프로젝트 추가" (Add project) 클릭
4. 프로젝트 이름 입력

```
our-taste-map
```

5. Google Analytics 설정

```
"Google Analytics 사용 설정" 토글
→ 끄기 (지금은 필요 없음)
→ "프로젝트 만들기" 클릭
```

6. 기다리기 (30초~1분)

```
프로젝트 생성 중...
→ "프로젝트가 준비되었습니다" 나오면 "계속" 클릭
```

✅ Firebase 프로젝트 생성 완료!

## 2단계: Firestore 데이터베이스 만들기

1. 왼쪽 메뉴에서 "빌드" → "Firestore Database" 클릭
2. "데이터베이스 만들기" 클릭
3. 보안 규칙 선택

```
○ 프로덕션 모드에서 시작 (권장)
● 테스트 모드에서 시작 ← 이거 선택!

→ "다음" 클릭
```

4. Cloud Firestore 위치 선택

```
asia-northeast3 (서울) ← 추천!
→ "사용 설정" 클릭
```

5. 기다리기 (1~2분)

```
Firestore 프로비저닝 중...
→ 완료되면 빈 데이터베이스 화면 보임
```

✅ Firestore 데이터베이스 생성 완료!

## 3단계: Firebase Storage 설정

1. 왼쪽 메뉴에서 "빌드" → "Storage" 클릭
2. "시작하기" 클릭
3. 보안 규칙 선택

```
● 테스트 모드에서 시작 ← 선택
→ "다음" 클릭
```

4. Cloud Storage 위치

```
asia-northeast3 (서울) ← 자동 선택됨
→ "완료" 클릭
```

✅ Firebase Storage 생성 완료!

## 4단계: 웹 앱 추가 및 API 키 받기

1. Firebase 프로젝트 개요 (홈) 화면으로

```
좌측 상단 "프로젝트 개요" 클릭
```

2. 웹 앱 추가

```
"</>" (웹) 아이콘 클릭
```

3. 앱 닉네임 입력

```
앱 닉네임: our-taste-map-web
□ Firebase Hosting 설정 (체크 안 함)
→ "앱 등록" 클릭
```

4. Firebase SDK 추가

```
화면에 코드가 나타남 (잠깐만 기다려주세요!)
```

중요! 이 화면 캡처하거나 복사해주세요:

```javascript
const firebaseConfig = {
  apiKey: "AIzaSy...",
  authDomain: "our-taste-map.firebaseapp.com",
  projectId: "our-taste-map",
  storageBucket: "our-taste-map.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456789:web:..."
};
```

→ "콘솔로 이동" 클릭
✅ Firebase 설정 정보 받기 완료!
