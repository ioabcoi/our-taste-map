# npx tailwindcss init -p 에러

## 🛠️ 해결 방법

### Option 1: node_modules/.bin 직접 실행 (추천)

```bash
.\node_modules\.bin\tailwindcss init -p
```

### Option 2: package.json에 스크립트 추가

- package.json 파일 열기
- "scripts" 부분에 추가:

```json
{
  "scripts": {
    "dev": "vite",
    "build": "tsc && vite build",
    "preview": "vite preview",
    "tailwind:init": "tailwindcss init -p"  // ← 이거 추가
  }
}
```

- 실행:

```bash
npm run tailwind:init
```

### Option 3: 수동으로 파일 생성

1. npx 안 써도 돼요! 제가 만들어드린 파일 그대로 쓰시면 됩니다!

2. 이미 만들어드린 파일:
- ✅ tailwind.config.js (이미 있음)
- postcss.config.js (이것만 추가하면 됨)

3. postcss.config.js 파일 생성:
our-taste-map 폴더에 postcss.config.js 파일 만들고 이렇게 넣으세요:

```javascript
export default {
  plugins: {
    tailwindcss: {},
    autoprefixer: {},
  },
}
```

### ✅ 그냥 이렇게 하세요! (제일 쉬움)

## Tailwind CSS v4 버전 문제
최신 버전이 설치되어서 그래요. v3로 다시 설치해야 해요!

## 🔧 해결 방법

1. 현재 Tailwind 삭제

```bash
npm uninstall tailwindcss
```

2. Tailwind v3 설치

```bash
npm install -D tailwindcss@3.4.1 postcss autoprefixer
```

3. postcss.config.js 파일 확인
이렇게 되어있는지 확인:

```javascript
export default {
  plugins: {
    tailwindcss: {},
    autoprefixer: {},
  },
}
```

4. 다시 실행

```bash
npm run dev
```
