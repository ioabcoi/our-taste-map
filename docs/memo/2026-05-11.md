## 질문

### 🔍 문제 1: Vercel에서 Firebase 추가 안 되는 이유

```
원인: 환경변수가 없어서!
.env.local은 로컬에만 있고, Vercel에는 없어요!
해결: Vercel에 환경변수 추가
```

#### 1. Vercel 대시보드 접속

```
https://vercel.com/dashboard
→ our-taste-map 프로젝트 클릭
```

#### 2. Settings → Environment Variables

```
Settings 탭 클릭
→ 왼쪽 메뉴에서 "Environment Variables" 클릭
```

#### 3. 환경변수 추가

```
Name: VITE_FIREBASE_API_KEY
Value: (Firebase에서 받은 API 키)
Environment: Production, Preview, Development 모두 체크
→ Save

이걸 6개 모두 추가:
- VITE_FIREBASE_API_KEY
- VITE_FIREBASE_AUTH_DOMAIN
- VITE_FIREBASE_PROJECT_ID
- VITE_FIREBASE_STORAGE_BUCKET
- VITE_FIREBASE_MESSAGING_SENDER_ID
- VITE_FIREBASE_APP_ID
```

#### 4. 재배포

```
Deployments 탭
→ 최신 배포 우측 ... 버튼
→ "Redeploy" 클릭
```

#### 5. 확인

```
https://our-taste-map.vercel.app
→ 이제 버튼 작동할 거예요!
```

### 🔍 문제 2: GitHub Pages 안 열리는 이유

```
원인: GitHub Pages는 설정 안 했어요!
지금 배포는:
✅ Vercel: our-taste-map.vercel.app (작동 중)
❌ GitHub Pages: ioabcoi.github.io/our-taste-map (설정 안 함)
```

GitHub Pages vs Vercel

|----|Vercel|GitHub Pages|
|설정|자동|수동 필요|
|SPA 라우팅|지원|불편|
|환경변수|지원|불가능|
|속도|빠름|느림|

→ Vercel만 쓰세요! GitHub Pages는 불필요해요

### Step 4: Map 컴포넌트 만들기 (20분)

```typescript
import { Map, MapMarker } from 'react-kakao-maps-sdk';

interface MapComponentProps {
  center?: {
    lat: number;
    lng: number;
  };
}

function MapComponent({ center = { lat: 37.5665, lng: 126.9780 } }: MapComponentProps) {
  return (
    <div className="card">
      <h3 className="text-lg font-semibold mb-4">🗺️ 카카오맵 테스트</h3>
      
      <Map
        center={center}
        style={{ width: '100%', height: '400px', borderRadius: '8px' }}
        level={3}
      >
        {/* 기본 마커 (서울시청) */}
        <MapMarker position={center} />
      </Map>
      
      <p className="text-sm text-gray-600 mt-2">
        📍 현재 위치: 서울시청
      </p>
    </div>
  );
}

export default MapComponent;
```
