## 5단계: 로컬 프로젝트에 Firebase SDK 설치

### 1. 터미널 열기

VSCode에서:

```
Ctrl + ` (백틱)
또는
상단 메뉴 → Terminal → New Terminal
```

프로젝트 폴더 확인:

```bash
# 현재 위치 확인
pwd  # Mac/Linux
cd   # Windows

# our-taste-map 폴더에 있어야 함
```

### 2. Firebase 설치

```bash
npm install firebase
```

기다리기: 10~30초

완료 메시지:

```
added 1 package, and audited 179 packages in 15s
```

✅ Firebase 설치 완료!

### 3. .env.local 파일 생성

프로젝트 루트에 .env.local 파일 만들기:

```
.env.local 템플릿 파일 >
```

이 파일을 our-taste-map/.env.local로 저장하고 아까 받은 Firebase config 값으로 교체하세요!

### 4. Firebase 초기화 파일 만들기

src/firebase.ts 파일 생성:

```
Src/firebase.ts - Firebase 초기화 >
```

### 5. 연결 테스트

src/App.tsx 수정해서 테스트:

```
App.tsx - Firebase 연결 테스트 >
```

### 6단계: 실제 데이터 테스트

간단한 테스트 컴포넌트 만들기

```
Src/components/FirebaseTest.tsx - 데이터 추가/조회 테스트 >
```

App.tsx에 추가하기

```
App.tsx - FirebaseTest 컴포넌트 추가 >
```
