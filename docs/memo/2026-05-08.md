### 테스트 실행

#### 1. 개발 서버 실행

```bash

```

#### 2. 브라우저에서 확인

```
http://localhost:5173
```

#### 3. 테스트 버튼 클릭

① "테스트 장소 추가" 버튼 클릭

```
→ "장소 추가 성공! 🎉" 알림 뜨면 성공!
```

② "장소 목록 불러오기" 버튼 클릭

```
→ 방금 추가한 장소가 목록에 표시됨
→ 이름, 별점, 리뷰 보임
```

#### 4. Firebase Console에서 확인

```
Firebase Console → Firestore Database
→ places 컬렉션 생김
→ 문서 1개 보임
→ 클릭하면 데이터 확인 가능
```

✅ 성공 조건:

- 알림창 뜸
- 목록에 표시됨
- Firebase Console에 데이터 보임

# 🗺️ 카카오맵 vs 구글맵 동시 지원

결론: 가능하지만 v2.0에서 하세요!

## 💡 추천 전략

v1.0: 카카오맵만

```
✅ 빠른 개발
✅ 국내 데이터 정확
✅ 무료
✅ 복잡도 낮음
```

v2.0: 지도 전환 기능 추가

```
[카카오맵] [구글맵] ← 토글 버튼

사용자가 선택!
- 국내 → 카카오맵
- 해외 → 구글맵
```

## 🤔 왜 v2.0에서?

복잡해지는 것들

### 1. 설정 파일

```javascript
// 카카오맵
<script src="kakao-map-sdk.js"></script>

// 구글맵
<script src="google-maps-sdk.js"></script>

// 충돌 가능성 있음
```

### 2. 컴포넌트 구조

```typescript
// 카카오맵
<KakaoMap />

// 구글맵
<GoogleMap />

// 공통 인터페이스 설계 필요
```

### 3. API 키 관리

```env
VITE_KAKAO_MAP_API_KEY=...
VITE_GOOGLE_MAP_API_KEY=...
```

### 4. 좌표 체계

```
카카오맵: WGS84 (latitude, longitude)
구글맵: WGS84 (lat, lng)

→ 비슷하지만 미묘하게 다름
```

### 5. 마커/InfoWindow 구조

```javascript
// 카카오맵
new kakao.maps.Marker({...})

// 구글맵
new google.maps.Marker({...})

// 완전히 다른 API
```

## 🎯 확장 가능하게 설계하는 법

v1.0에서 미리 준비

### 1. 지도 컴포넌트 추상화

```typescript
// src/components/Map/MapContainer.tsx
interface MapContainerProps {
  places: Place[];
  center: { lat: number; lng: number };
  onMarkerClick: (place: Place) => void;
}

function MapContainer({ places, center, onMarkerClick }: MapContainerProps) {
  // 나중에 여기서 카카오/구글 선택
  return <KakaoMapView places={places} center={center} />;
}
```

### 2. 공통 타입 정의

```typescript
// src/types/map.ts
export interface Coordinates {
  latitude: number;
  longitude: number;
}

export interface Place {
  id: string;
  name: string;
  coordinates: Coordinates;
  // ...
}
```

```typescript
// Firestore 데이터
```

## 🎨 v2.0 UI 예상

```typescript
function MapContainer() {
  const [mapType, setMapType] = useState<'kakao' | 'google'>('kakao');

  return (
    <div>
      {/* 지도 선택 버튼 */}
      <div className="flex gap-2 mb-4">
        <button 
          onClick={() => setMapType('kakao')}
          className={mapType === 'kakao' ? 'active' : ''}
        >
          카카오맵 (국내)
        </button>
        <button 
          onClick={() => setMapType('google')}
          className={mapType === 'google' ? 'active' : ''}
        >
          구글맵 (해외)
        </button>
      </div>

      {/* 지도 표시 */}
      {mapType === 'kakao' ? (
        <KakaoMapView places={places} />
      ) : (
        <GoogleMapView places={places} />
      )}
    </div>
  );
}
```
