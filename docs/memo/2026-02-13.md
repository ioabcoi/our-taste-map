## 📊 와이어프레임 vs 다른 것들

```
[와이어프레임]     [목업]          [프로토타입]
    뼈대      →    디자인 적용   →   실제 동작
  (흑백 위주)     (색상/폰트)      (클릭 가능)
```

### 예시로 보면

**와이어프레임 (오늘 할 것)**
```
┌─────────────────┐
│ [로고] [메뉴]   │ ← 회색 박스
├─────────────────┤
│ [검색창]        │
│ [필터] [필터]   │
├─────────────────┤
│                 │
│   [지도 영역]   │ ← "여기 지도"라고만 표시
│                 │
└─────────────────┘
    [+ 버튼]
```

**목업 (나중에)**
```
┌─────────────────┐
│ 🗺️ Our-Taste   │ ← 실제 색상, 폰트
├─────────────────┤
│ 🔍 검색...      │
│ [맛집][카페]    │ ← 예쁜 버튼
├─────────────────┤
│                 │
│  실제 지도 이미지│ ← 카카오맵처럼
│                 │
└─────────────────┘
```

## 🎯 와이어프레임의 목적

### 1. **레이아웃 확정**
- "버튼을 어디에 둘까?"
- "검색창은 위? 아래?"
- "모달 크기는?"

### 2. **사용자 플로우 검증**
- "여기 클릭하면 여기로 가는 게 맞나?"
- "단계가 너무 많지 않나?"

### 3. **개발 전 합의**
- "이렇게 만드는 거 맞죠?"
- 나중에 "이거 아닌데..." 방지

### 4. **빠른 수정**
- 색칠하기 전에 뼈대만 그려서
- 마음에 안 들면 바로 고치기

## 📝 와이어프레임 작성 방법

### Option 1: 손그림 (제일 빠름!)
```
종이에 네모 박스 그리고
→ 사진 찍어서
→ main-map.png로 저장
```

**장점:**
- ✅ 제일 빠름
- ✅ 생각나는 대로 쭉쭉
- ✅ 도구 필요 없음

**단점:**
- ❌ 수정 어려움
- ❌ 깔끔하진 않음

---

### Option 2: Figma (추천!)
```
Figma (무료) 사용
→ 간단한 도형으로 배치
→ PNG 내보내기
```

**장점:**
- ✅ 깔끔함
- ✅ 수정 쉬움
- ✅ 나중에 디자인도 여기서
- ✅ 남편이랑 같이 보기 좋음

**단점:**
- ❌ 도구 익히는 시간 필요

---

### Option 3: Balsamiq / Excalidraw
```
전용 와이어프레임 툴
→ 손그림처럼 보이게
→ 빠르고 직관적
```

**추천 도구:**
- **Excalidraw** (https://excalidraw.com) - 무료, 손그림 스타일
- **Balsamiq** - 유료, 전문적
- **Whimsical** - 무료 플랜 있음

## 🚀 실전 작업 순서

### 방법 1: 손그림 → 디지털

```
1. 종이에 연필로 러프 스케치
   - A4 용지 준비
   - 네모 박스로 레이아웃 그리기
   - 버튼, 텍스트 위치 표시
   ↓
2. "이 정도면 되겠다" 싶으면
   - 스마트폰으로 사진 촬영
   - 또는 스캔
   ↓
3. (선택) Figma에서 깔끔하게 정리
   - 손그림 보고 다시 그리기
   - 도형 도구로 깔끔하게
   ↓
4. PNG로 저장
   ↓
5. /docs/planning/wireframes/ 에 저장
   ↓
6. Git 커밋
```

### 방법 2: 바로 Figma

```
1. Figma 접속 (figma.com)
   ↓
2. 새 파일 생성
   - "Our-Taste-Map Wireframes"
   ↓
3. Frame 생성 (F)
   - Mobile: 375 x 812
   - Desktop: 1440 x 900
   ↓
4. 도형으로 레이아웃 구성
   - Rectangle (R): 박스, 버튼
   - Text (T): 텍스트, 라벨
   - Line (L): 구분선
   ↓
5. Export as PNG
   ↓
6. 저장 및 커밋
```

## 🛠️ Figma 간단 사용법

### 기본 단축키
```
F - Frame 생성 (화면 전체)
R - Rectangle (네모 박스)
T - Text (텍스트)
O - Circle (동그라미)
L - Line (선)
V - 선택 도구
```

### 작업 흐름
```
1. Frame 만들기 (F)
   → 375x812 (모바일) 또는 1440x900 (데스크톱)

2. Rectangle로 영역 나누기 (R)
   → Header, Filter, Content, Footer

3. Text로 라벨 추가 (T)
   → [검색창], [버튼], 등등

4. 정렬 도구 사용
   → 오른쪽 패널에서 정렬 버튼

5. Export
   → 우측 하단 Export → PNG → Export
```

### 꿀팁
- **Auto Layout** 사용하면 자동 정렬
- **Components** 만들어서 재사용 (버튼 등)
- **Ctrl + D** 복제
- **Shift + 드래그** 일직선 이동
