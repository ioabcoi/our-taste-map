### Option 1: 와이어프레임 그리기 (추천!)

#### 📋 진행 방식

##### 1단계: 도구 선택

```
어떤 방법으로 그릴까요?

A. Figma (추천!)

장점: 깔끔, 수정 쉬움, 남편이랑 공유 편함
단점: 처음이면 약간 배워야 함
시간: 화면당 30분~1시간

B. 손그림 + 사진

장점: 제일 빠름, 자유로움
단점: 수정 어려움, 덜 깔끔
시간: 화면당 10~20분

C. Excalidraw (중간)

장점: 손그림처럼 쉬움, 웹에서 바로
단점: Figma보다 기능 적음
시간: 화면당 20~30분
```

##### 2단계: 그릴 화면 순서
```
1. main-map.png (지도 뷰) ← 제일 중요!
2. detail-modal.png (상세 모달)
3. place-form-modal.png (등록 모달)
4. list-view.png (리스트 뷰)
```
